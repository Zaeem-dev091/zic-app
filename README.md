## 📱 About The App

ZIC is a mobile mining application where users earn ZIC coins through
daily activity. Users can transfer coins, withdraw earnings, collect
daily rewards and spin the wheel for bonus coins.

---

## ✨ Features

- 🔐 User Authentication (Login & Signup via REST API)
- ⛏️ Coin Mining — earn ZIC coins daily
- 💸 Coin Transfer — send coins to other users
- 🏧 Coin Withdrawal — withdraw your earned coins
- 🎁 Daily Rewards — collect bonus coins every day
- 🎡 Spin Wheel — claim extra rewards by spinning
- 📊 Wallet Dashboard — track your balance and history
- 🏆 Level System — level up as you mine more

## 🚀 Getting Started

### Prerequisites
- Flutter SDK installed
- Android Studio or VS Code
- A device or emulator

### Installation

1. Clone the repo
```bash
   git clone https://github.com/Zaeem-dev091/zic-app.git
```

2. Go to project folder
```bash
   cd zic-app
```

3. Install dependencies
```bash
   flutter pub get
```

4. Run the app
```bash
   flutter run
```
